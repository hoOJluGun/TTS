## Что внутри

### 🎨 art_queue.csv (обновлён v0.2)
- Киноплёнка: Kodak Portra 400, Fujifilm Eterna, Ilford
- Свет: volumetric light rays, chiaroscuro
- Оптика: 35mm, 50mm, 85mm lenses
- Формат: 16:9

### 🎙 tts_queue_ssml.csv (новый)
- SSML-теги: `<break>`, `<prosody>`
- Настройки: rate (скорость), pitch (высота)
- Эмоции: вздохи, паузы, ускорения
