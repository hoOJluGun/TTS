## Использование

### 1. Генерация картинок
Используй `visual/art_queue.csv` в:
- Midjourney
- Leonardo.ai  
- Stable Diffusion
- DALL-E

### 2. Озвучка
Используй `tts_queue_ssml.csv` с:
- ElevenLabs
- Edge TTS
- Coqui TTS

### 3. Слушать
Открой `audio-player.html` в браузере
